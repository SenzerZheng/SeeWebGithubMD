### 空格

quad 中文：四

| 两个quad空格 | a \qquad b | 两个*m*的宽度  |
| ------------ | ---------- | -------------- |
| quad空格     | a \quad b  | 一个*m*的宽度  |
| 大空格       | a\ b       | 1/3*m*宽度     |
| 中等空格     | a\;b       | 2/7*m*宽度     |
| 小空格       | a\,b       | 1/6*m*宽度     |
| 没有空格     | ab         |                |
| 紧贴         | a\!b       | 缩进1/6*m*宽度 |
